Algebra

Introduction

Algebra is the study of variables and their relationships.

It is a fundamental area of mathematics that has numerous applications in science, engineering, and other fields.

Equations and Inequalities

An equation is a statement that two expressions are equal.

For example, 2x + 3 = 5 is an equation.

An inequality is a statement that one expression is greater than or less than another expression.

For example, 2x + 3 > 5 is an inequality.

Functions and Graphs

A function is a relation between a set of inputs and a set of possible outputs.

For example, f(x) = 2x + 3 is a function.

A graph is a visual representation of a function or relation.

For example, the graph of f(x) = 2x + 3 is a line.
